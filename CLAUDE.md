# Doc Workspace — agent rules

Full project orientation and conventions are in @AGENTS.md (imported below) —
that file is the source of truth. Key rules in brief:

- **App Router only** — never create a `pages/` directory; shared components go in `app/components/`.
- **This Next.js version has breaking changes** — read the relevant guide in `node_modules/next/dist/docs/` before writing code.
- **No new dependencies** without asking first.
- **No secrets or API keys** in source files.
- The person directing this is a **non-coder** — explain design and technical decisions in plain language.
